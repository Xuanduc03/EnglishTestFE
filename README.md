English Web Frontend
Frontend cho hệ thống luyện thi TOEIC được xây dựng bằng React + Vite + TypeScript + Ant Design.

1. Yêu cầu hệ thống
Trước khi chạy project, cần cài đặt:
Node.js >= 18
npm hoặc yarn
Git
Kiểm tra phiên bản:
node -v
npm -v
git --version

2. Clone project từ GitHub
git clone [https://github.com/<your-username>/toeic-web-fe.git](https://github.com/Xuanduc03/EnglishTestFE)
Di chuyển vào thư mục project:
cd toeic-web-fe


3. Cài đặt dependencies
Chạy lệnh sau để cài toàn bộ thư viện:
npm install

Project sử dụng các thư viện chính:
- React 19
- Vite
- TypeScript
- Ant Design
- Zustand (state management)


4. Chạy project ở môi trường development
Chạy server local:
npm run dev

Chart.js / Recharts (chart)

TipTap (rich text editor)
